Description

Description

Research rover built for a proposal on exploration of the lunar Copernicus craters. Man in the loop drive system, with signal degradation capabilities for simulating light time delays, bandwidth issues, etc… Built to run on a mobile polarized projection setup, providing the driver with a 3D view while also being easily deployable from a large vehicle.
